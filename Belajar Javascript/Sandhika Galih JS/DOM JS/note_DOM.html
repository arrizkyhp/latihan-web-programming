NodeList vs. HTMLCollection

NodeList

Ketika kita memilih lebih dari satu node

HTMLCollection

Kumpulan Node Khusus yang tipenya element html saja


Perbedaan ini akan menentukan cara mengelolanya.

  • Keduanya merupakan kumpulan node
  • Struktur datanya mirip array
  • nodeList dapat berisi node apapun
  • HTMLCollection harus berisi elemen HTML dapat berisi node apapun
  • HTMLCollectionbersifat live sedangkan nodeList tidak

Strukture Hierarki DOM Tree

Konsep Hierarki DOM tree mirip silsilah keluarga, adanya parent & child

  • Root Node
    • node yang menjadi sumber dari semua node lain di DOM
    • secara default node tersebut adalah document
  • Parent Node
    • node yang berada 1 tingkat di atas node yang lain
    • body adalah parent dari h1, p,div dan a
  • Child Node
    • node yang berada 1 tingkat di bawah node yang lain
    • h1 adalah child dari body
    • Children adalah child node yang bertipe html
    • Siblings adalah child node yang memiliki parent yang sama

Memilih element-element pada DOM menggunakan Javascript.

DOM Selection merupakan tahap awal ketika ingin melakukan manipulasi atau penelusuran pada DOM.

getElementById & getElementsByTagName merupakan DOM selection yang tercepat, jadi pastikan dulu bisa gunakan dua selection tersebut.

DOM Selection Method

  • getElementById() = mencari element yang ada di DOM kita berdasarkan id / hanya satu element
  • getElementsByTagName() = mencari element-element yang memiliki nama tag tertentu
  • getElementsByClassName() = mencari element-element yang memiliki nama class tertentu
  • querySelector()
  • querySelectorAll() = mencari element berdasarkan selectornya
Method Hasil
getElementById() element
getElementsByTagName() HTMLCollection
getElementsByClassName() HTMLCollection
querySelector() element
querySelectorAll() nodeList

    const namaVariable = document.getElementById('namaId');

    // Inisialisasi jika ingin mengganti warna background
    namaVariable.style.backgroundColor = 'Blue';

DOM Manipulation Method

Manipulasi Element

Element yang sudah diseleksi sebelumnya dapat dimanipulasi, contoh mengambil paragraf lalu mengubah isinya

Beberapa methodnya yaitu:

  • element.innerHTML => merubah isi dari sebuah tag
  • element.style.<properti> => untuk merubah style/css
  • element.setAttribute() => untuk memanipulasi atribut dalam pada element. Menimpa attribute yang sudah ada.
  • element.classList => memanipulasi class
    • element.classList.add() => tambah kelas baru
    • element.classList.remove() => hapus kelas yang ada
    • element.classList.toggle() => menambahkan dan menghilangkan kelas
    • element.classList.item() => mengetahui kelas tertentu dalam sebuah elemen
    • element.classList.contains() => mengecek dalam sebuah apakah memiliki kelas tertentu
    • element.classList.replace() => mengganti kelas yang ada dengan yang baru
  • dsb...

    namaValue.innerHTML ='<h1> Hello Universe </h1>'

element.style.<properti>


    namaValue.style.BackgroundColor = 'gray';

    namaValue.setAttribute('class','nama-classnya');

element.classList()


    // menambahkan class baru
    namaValue.classList.add('namaClassBaru');

    namaValue.classList.remove('namaClassYangInginDiHapus');

    // menambahkan & menghapus class / On Off
    namaValue.classList.toggle('namaClass');

    namaValue.classList.item(1);

    namaValue.classList.contains('namaClassYangDicari');

Manipulasi Node

Ketika kita ingin menyisipkan element baru pada halaman, contoh kita punya 3 paragraf ingin menambahkan menjadi 4 atau menghilangkan sebuah element

Memungkinkan kita untuk menambahkan, menghapus & mengganti node pada DOM

Beberapa Methodnya yaitu:

  • document.createElement() => membuat Element
  • document.createTextNode() => menuliskan apa di dalam element
  • node.appendChild() => menyimpan element dan isinya di bagian akhir
  • node.insertBefore() => menyimpan element dan isinya di bagian awal
  • parentNode.removeChild()
  • parentNode.replaceChild()
  • dsb...
