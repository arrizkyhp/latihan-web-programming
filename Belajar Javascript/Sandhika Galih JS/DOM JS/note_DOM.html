NodeList vs. HTMLCollection

NodeList

Ketika kita memilih lebih dari satu node

HTMLCollection

Kumpulan Node Khusus yang tipenya element html saja


Perbedaan ini akan menentukan cara mengelolanya.

  • Keduanya merupakan kumpulan node
  • Struktur datanya mirip array
  • nodeList dapat berisi node apapun
  • HTMLCollection harus berisi elemen HTML dapat berisi node apapun
  • HTMLCollectionbersifat live sedangkan nodeList tidak

Strukture Hierarki DOM Tree

Konsep Hierarki DOM tree mirip silsilah keluarga, adanya parent & child

  • Root Node
    • node yang menjadi sumber dari semua node lain di DOM
    • secara default node tersebut adalah document
  • Parent Node
    • node yang berada 1 tingkat di atas node yang lain
    • body adalah parent dari h1, p,div dan a
  • Child Node
    • node yang berada 1 tingkat di bawah node yang lain
    • h1 adalah child dari body
    • Children adalah child node yang bertipe html
    • Siblings adalah child node yang memiliki parent yang sama

Memilih element-element pada DOM menggunakan Javascript.

DOM Selection merupakan tahap awal ketika ingin melakukan manipulasi atau penelusuran pada DOM.

getElementById & getElementsByTagName merupakan DOM selection yang tercepat, jadi pastikan dulu bisa gunakan dua selection tersebut.

DOM Selection Method

  • getElementById() = mencari element yang ada di DOM kita berdasarkan id / hanya satu element
  • getElementsByTagName() = mencari element-element yang memiliki nama tag tertentu
  • getElementsByClassName() = mencari element-element yang memiliki nama class tertentu
  • querySelector()
  • querySelectorAll() = mencari element berdasarkan selectornya
Method Hasil
getElementById() element
getElementsByTagName() HTMLCollection
getElementsByClassName() HTMLCollection
querySelector() element
querySelectorAll() nodeList

getElementById()


    const namaVariable = document.getElementById('namaId');

    // Inisialisasi jika ingin mengganti warna background
    namaVariable.style.backgroundColor = 'Blue';

getElementsByTagName()


    const namaVariable = document.getElementsByTagName('namaId');

    // Inisialisasi
    // Cara 1
    p[3].style.backgroundColor = 'Salmon';
    // Cara 2 di variable
    const p = document.getElementsByTagName('p')[3];

getElementsByClassName()


    const namaVariable = document.getElementsByClassName('namaClass')[0];

    // Inisialisasi sama dengan getElementsByTagName()

querySelector()

Selector => selector css


    const namaVariable = document.querySelector('#namaIdSelectorCss h1');

querySelectorAll()


    const namaVariable = document.querySelector('#namaIdSelectorCss h1');

DOM Manipulation Method

Manipulasi Element

Element yang sudah diseleksi sebelumnya dapat dimanipulasi, contoh mengambil paragraf lalu mengubah isinya

Beberapa methodnya yaitu:

  • element.innerHTML => merubah isi dari sebuah tag
  • element.style.<properti> => untuk merubah style/css
  • element.setAttribute() => untuk memanipulasi atribut dalam pada element. Menimpa attribute yang sudah ada.
  • element.classList => memanipulasi class
    • element.classList.add() => tambah kelas baru
    • element.classList.remove() => hapus kelas yang ada
    • element.classList.toggle() => menambahkan dan menghilangkan kelas
    • element.classList.item() => mengetahui kelas tertentu dalam sebuah elemen
    • element.classList.contains() => mengecek dalam sebuah apakah memiliki kelas tertentu
    • element.classList.replace() => mengganti kelas yang ada dengan yang baru
  • dsb...

element.innerHTML


    namaValue.innerHTML ='<h1> Hello Universe </h1>'

element.style.<properti>


    namaValue.style.BackgroundColor = 'gray';

element.setAttribute()


    namaValue.setAttribute('class','nama-classnya');

element.classList()


    // menambahkan class baru
    namaValue.classList.add('namaClassBaru');

    // menghapus class
    namaValue.classList.remove('namaClassYangInginDiHapus');

    // menambahkan & menghapus class / On Off
    namaValue.classList.toggle('namaClass');

    // mengetahui class dalam sebuah elemen
    namaValue.classList.item(1);

    // mencari class dalam element => output berupa boolean
    namaValue.classList.contains('namaClassYangDicari');

    // mengganti class dalam element
    namaValue.CLassList.replace('namaClassLama', 'namaClassBaru');

Manipulasi Node

Ketika kita ingin menyisipkan element baru pada halaman, contoh kita punya 3 paragraf ingin menambahkan menjadi 4 atau menghilangkan sebuah element

Memungkinkan kita untuk menambahkan, menghapus & mengganti node pada DOM

Beberapa Methodnya yaitu:

  • document.createElement() => membuat Element
  • document.createTextNode() => menuliskan apa di dalam element
  • node.appendChild() => menyimpan element dan isinya di bagian akhir
  • node.insertBefore() => menyimpan element dan isinya di bagian awal
  • parentNode.removeChild()
  • parentNode.replaceChild()
  • dsb...

    const variabelElement = document.createElement('h1');

document.createTextNode()


    const variabelText = document.createTextNode('Hello World');

node.appendChild()


    variabelElement.appendChild(variabelText);

node.insertBefore()


    // buat elemen barunya dengan cara diatas

    // tangkap parent
    const parentElement = document.querySelector('ul');
    // tangkap elemen setelahnya
    const liAfter = parentElement.querySelector('li:nth-child(2)');

    // inisialisasi
    parentElement.insertBefore(variabelElementBaru, liAfter);

parentNode.removeChild()


    // tangkap parent
    const parentSectionA = document.getElementById('a');
    // tangkap elemen yang akan di remove
    const link = document.getElementsByTagName('a')[0];

    // hapus
    parentSectionA.removeChild(link);

parentNode.replaceChild()


    // Node Lama
    // tangkap parent
    const parentSectionB = document.getElementById('b');
    // tangkap elemen yang akan di replace
    const p4 = parentSectionB.querySelector('p');

    // buat elemen baru
    const h2Baru = document.createElement('h2');
    // isi text
    const h2Text = document.createTextNode('Judul Section B');
    // masukan text ke elemen
    h2Baru.appendChild(h2Text);

    // replace
    parentSectionB.replaceChild(h2Baru, p4);
DOM Events

ðŸš€ File: dom_eventHandler.html & domEventHandler.js

Event pada javascript merepresentasikan sebuah kejadian yang terjadi di dalam DOM, kejadian tersebut bisa dilakukan oleh user (mouse event, keyboard event, dll), ataupun dilakukan secara otomatis oleh API (animasi selesai dijalankan,, halaman selesai di load, dll).

Daftar Event lengkap

https://developer.mozilla.org/en-US/docs/Web/Events

Cara 'mendengarkan' event

  • Event Handler => Cara lama
    • Inline HTML attribute => Langsung ditanam di HTML
    • Element method
  • addEventListener()

Perbedaan dari kedua event diatas adalah ketika kita memberikan lebih dari satu perubahan pada sebuah event. Jika kita menggunakan Event Handler perubahan yang terakhir kita lakukan akan menimpa perubahan sebelumnya, sedangkan addEventListener akan menambah perubahannya.

Event Handler

on<event> => onclick, onchange, onload, dll

Inline HTML attribute

Inline HTML Attribute tidak disarankan karena mencampurkan html dan js 


    // panggil element
    const p3 = document.querySelector('.p3');

    // buat function
    function ubahWarna() {
    p3.style.backgroundColor = 'salmon';
    }

    // buat onclick="ubahWarna()" di element .p3
    <p class="p3" onclick="ubahWarna()">paragraf 3</p>

Element method

hampir sama dengan Inline HTML attribute dibuat function, tetapi perbedaanya element method dibuat hanya di Javascript saja


    // panggil element
    const p2 = document.querySelector('.p2');

    // buat function
    function ubahWarna2() {
    p2.style.backgroundColor = 'salmon';
    }

    // inisialisasi
    p2.onclick = ubahWarna2;

addEventListener()


    // panggil element
    const p4 = document.querySelector('section#b p')

    // event
    p4.addEventListener('click', function() {
    // panggil parent
    const catchUl = document.querySelector('section#b ul');
    // buat elemen dan text
    const makeList = document.createElement('li');
    const textList = document.createTextNode('item baru');
    makeList.appendChild(textList);

    // inisialisasi
    catchUl.appendChild(makeList);
    });

preventDefault()

Mencegah aksi default pada elemen, contohnya <a> defaultnya membuka link, hal itu dapat diubah dengan preventDefault()


    event.addEventListener('click', (e) => {
    e.preventDefault();
    });
DOM Traversal

ðŸš€ File: dom_traversal.html & domTraversal.js

Penelusuran DOM

DOM Traversal Method

  • parentNode => ambil Parent Node
  • parentElement => ambil Parent Element
  • nextSibling => ambil "saudara kandung" dari elemen yang kita pilih setelahnya
  • nextElementSibling => sama dengan nextSibling tetapi ini elemen saudaranya
  • previousSibling => ambil "saudara kandung" dari elemen yang kita pilih sebelumnya
  • previousElementSibling => sama dengan previousSibling tetapi ini elemen saudaranya
Method Hasil
parentNode node
parentElement element
nextSibling node
nextElementSibling element
previousSibling node
previousElementSibling element

parentNode


    const nama = document.querySelector('.nama');

    console.log(nama.parentNode);
    // output <div class="card" ....

parentElement


    const nama = document.querySelector('.nama');

    console.log(nama.parentElement);
    // output <div class="card" ....

nextSibling

tidak direkomendasikan menggunakan ini, sebaiknya gunakan nextElementSibling 


    const nama = document.querySelector('.nama');

    console.log(nama.nextSibling);
    // #text | karena di html ada Enter antara nama dan telp

nextElementSibling


    const nama = document.querySelector('.nama');

    console.log(nama.nextElementSibling);
    // <span class="telp" | mengabaikan enter

previousSibling dan previouElementSibling sama penulisan kodenya dengan nextSibling dan nextElementSibling.
