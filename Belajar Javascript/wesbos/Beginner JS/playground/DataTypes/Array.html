Array

What is Array??

An array is an ordered list of values, or a collection of elements (values or variables) identified by an index or key.

Array is use for holding a list of item where the order matters

  • Each things inside an array is called an item.
  • Position in Array called index or key.
  • The number item inside an array called a length.
  • Each item inside array can be any type, can be string,number,boolean,etc.
  • While { } for object, the [ ] is for Array.
  • There are no key in array.
  • Array are Zero-based numbering, its mean start counting at zero => 0,1,2,3.

How to use??

JavaSCript
const names = ['arrizy', 'hasya', 'pratama'];

If you want to access thing / item inside an array you use index:

JavaSCript
console.log(names[1]); // O: hasya

Check how many item inside an array:

JavaSCript
console.log(names.length); // O: 3

Clone an Array using [...] / Spread


When you try to copy it using the = it will only copy the reference to the original array and not the value of the array. To create a real copy of an array, you need to copy over the value of the array under a new value variable. That way this new array does not reference to the old array address in memory.
