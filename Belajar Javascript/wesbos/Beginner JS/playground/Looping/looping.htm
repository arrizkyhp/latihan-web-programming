Looping

What is Looping??

Loops offer a quick and easy way to do something repeatedly.

The most common way is looping over an array

How to use??

forEach()

forEach() is handy for when you want to loop over some data and do something with each piece of data.

JavaSCript
const seblak = ['baso', 'kerupuk', 'usus', 'tulang', 'mie'];

seblak.forEach(function);

map()

map() is like a machine in a factory, it takes in data, performs an operation and spit it out on the other side.

map() can also be use in for any type of data

JavaSCript
const nama = ['bella', 'casandra', 'mortimer'];

const goth = nama.map((event) => `${event} goth`);
console.log(goth); // Â ["bella goth", "casandra goth", "mortimer goth"]

you can also do a chain map.

JavaSCript
function kapital(word) {
  return word[0].toUpperCase() + word.slice(1); }

const goth = nama.map(kapital).map((event) => `${event} Goth`);
console.log(goth); // Â ["Bella Goth", "Casandra Goth", "Mortimer Goth"]

filter()

filter() method creates a new array with all elements that pass the test implemented by the provided function.

the different between filter() and find()

  • filter() output are Array & search through all the elements
  • find() output are Object & search through all the child elements only.

JavaSCript
const keluarga = [
{ nama: 'Arrizky', umur: 24 },
{ nama: 'Luisa', umur: 20 },
{ nama: 'Lili', umur: 15 }, ];

const lebih18 = keluarga.filter((orang) => orang.umur > 18);
// {nama: "Arrizky", umur: 24}
// {nama: "Luisa", umur: 20}
